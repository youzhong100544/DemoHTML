方法一

使用 border 加 border-radius，在使用 background 隐藏内容

border-radius: 50%;
background: transparent;

position: relative;
}

.learn-01-01-contrast {
position: absolute;
top: 0;
left: 0;

width: 100px;
height: 100px;
border: 10px solid red;

/* 设置背景图片占满div且保持原有比例 */
background: url("../image/0101.png") no-repeat center / 100% 100%;

等效上面
background: url("../image/0101.png") no-repeat center;

}

</div>

</div>

1、绿色的为案例

CSS 画一个半圆环

CSS 的 clip-path 属性是 clip 属性的升级版，
它们的作用都是对元素进行 “剪裁”，
不同的是 clip 只能作用于 position 为 absolute 和 fixed 的元素且剪裁区域只能是正方形，
而 clip-path 更加强大，可以以任意形状去裁剪元素，且对元素的定位方式没有要求。

基于这样的特性，clip-path 常用于实现一些炫酷的动画效果。

style="
 width: 50px;
 height: 50px;
 border: 13px solid #0c73fe;
 border-radius: 50px;
 clip-path: polygon(100% 50%, 0 50%, 0 0, 100% 0);
"

代码说明

1、红色边框的 div 为参照 div
2、绿色的为案例
3、外层 div 是把参照 div 和 案例 div 叠加，方便展示效果

style="
 width: 50px;
 height: 50px;
 border: 13px solid #0c73fe;
 border-radius: 50px;
 clip-path: polygon(0 50%, 100% 50%, 100% 100%, 0 100%);
"

代码说明

1、红色边框的 div 为参照 div
2、绿色的为案例
3、外层 div 是把参照 div 和 案例 div 叠加，方便展示效果
